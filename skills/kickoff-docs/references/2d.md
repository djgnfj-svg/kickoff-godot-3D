# kickoff-docs: 2D 게임 부록

## 코어 버브 예시
- 달리기·점프 (플랫포머)
- 슬라이딩 / 대시 (회피·이동)
- 타격 / 슬래시 (근접 공격)
- 쏘기 (탑다운 슈터·사이드뷰 슈터)
- 짓기 (블록 쌓기·건물 배치, 탑다운/쿼터뷰)
- 캐기·수확 (자원 채집)
- 굴리기·당기기 (퍼즐)

## 시점 어휘
- 탑다운 / 사이드뷰 (횡스크롤) / 쿼터뷰 (아이소·픽셀) / 정면 (비주얼노벨식) / 보드뷰

## M0 Walking Skeleton 2D 예시
- 단색 사각형 캐릭터 (`Sprite2D` + 단색 PNG 또는 `ColorRect`)
- 단색 바닥 (`StaticBody2D` + `CollisionShape2D` 또는 `TileMap` 한 줄)
- `Camera2D` 가 캐릭터를 따라가기
- 입력: 좌우 이동 + 점프 (사이드뷰) 또는 4/8방향 이동 (탑다운)
- 종료 조건: 캐릭터가 바닥 위에서 이동·점프하고, 시점이 따라가고, 충돌이 일어남
