# kickoff-docs: 3D 게임 부록

## 코어 버브 예시
- 쏘기 (1인칭 슈터·3인칭 슈터)
- 베기 (검·근접 무기)
- 잡기 (그랩·인질·물건 운반)
- 회피 (구르기·대시·점프)
- 날기 (글라이더·제트팩·차량)
- 운전·조종 (차량·우주선)

## 시점 어휘
- 1인칭 / 3인칭 어깨너머 / 3인칭 쿼터뷰 / 탑다운 3D / 고정 카메라

## M0b Walking Skeleton 3D 예시
- 회색 박스 캐릭터 (`MeshInstance3D` + `BoxMesh`)
- 회색 평면 바닥 (`MeshInstance3D` + `PlaneMesh` + `StaticBody3D`)
- `Camera3D` 가 캐릭터를 따라가기 (1인칭이면 머리 위치, 3인칭이면 어깨 너머 일정 거리)
- `DirectionalLight3D` 1개
- 입력: WASD 이동 + 마우스 룩 (1인칭) 또는 WASD 이동 + 카메라 자동 (3인칭)
- 종료 조건: 캐릭터가 바닥 위에서 8방향 이동, 시점이 따라가고, 충돌이 일어남
