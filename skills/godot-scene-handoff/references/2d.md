# godot-scene-handoff: 2D 차원별 디테일

## 루트·카메라·조명
- 씬 루트: `Node2D`
- 카메라: `Camera2D` (Current 활성)
- 조명: 옵션 — `CanvasModulate` (전체 색조 변경), `Light2D` (스폿/포인트 라이트)

## 물리 노드
- 캐릭터: `CharacterBody2D` + `CollisionShape2D` + 자식 `Sprite2D` 또는 `AnimatedSprite2D`
- 동적 오브젝트: `RigidBody2D` + `CollisionShape2D`
- 정적 지형: `StaticBody2D` + `CollisionShape2D` (또는 `TileMap` 자체 collision)
- 영역 트리거: `Area2D` + `CollisionShape2D`

## Rendering 설정 (Project Settings → Display → Window)
- `stretch_mode`: `canvas_items` 권장 (해상도 비율 유지하며 캔버스 스케일)
- `stretch_aspect`: `keep` (검은 띠 허용) 또는 `expand` (확장)
- 픽셀아트: `texture_filter = Nearest` (Project Settings → Rendering → Textures → Default Texture Filter)
- 픽셀 퍼펙트: viewport size를 픽셀 해상도(예: 320×180)로 고정 후 `canvas_items` 스트레치

## 좌표계
- Godot 2D는 **Y-down** (스크린 좌표계). 3D와 부호 반대에 주의
- 회전 단위는 라디안. UI에서는 도(degree) 표기 흔함

## 입력 (2D 특이사항)
- `Input.get_vector("ui_left", "ui_right", "ui_up", "ui_down")` 직접 이동에 사용
- 마우스 클릭 좌표: `get_global_mouse_position()` → 직접 게임 좌표로 사용 (3D처럼 raycast 변환 불필요)
