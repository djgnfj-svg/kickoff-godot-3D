# godot-scene-handoff: 3D 차원별 디테일

## 루트·카메라·조명
- 씬 루트: `Node3D`
- 카메라: `Camera3D` (Current 활성)
- 조명: `DirectionalLight3D` 1개 + 필요 시 `OmniLight3D` / `SpotLight3D`
- 환경: `WorldEnvironment` (sky·ambient)

## 물리 노드
- 캐릭터: `CharacterBody3D` + `CollisionShape3D` + 자식 `MeshInstance3D`
- 동적 오브젝트: `RigidBody3D` + `CollisionShape3D`
- 정적 지형: `StaticBody3D` + `CollisionShape3D`
- 영역 트리거: `Area3D` + `CollisionShape3D`

## Rendering 설정
- 렌더러: Forward+ (데스크톱 고품질) / Mobile (모바일·Web) / Compatibility (저사양·웹)
- 선택 가이드: 플랫폼 + 시각 품질 요구에 따름. 의심 시 Forward+로 시작
- HDR / glow / SSR 등은 v0.x 초기엔 비활성화 (성능 안전)

## 좌표계
- Godot 3D는 **Y-up, -Z forward** (오른손 좌표계)
- 모델 import 시 forward axis 주의 (Blender·glTF 기본은 -Z forward 일치)

## 입력 (3D 특이사항)
- `Input.get_vector("ui_left", "ui_right", "ui_up", "ui_down")` 후 카메라 basis로 회전
- 마우스 룩(1인칭/3인칭): `_input(event)` + `Camera3D.rotate_*` + `Input.MOUSE_MODE_CAPTURED`
