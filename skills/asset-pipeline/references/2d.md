# asset-pipeline: 2D placeholder 카탈로그

2D 프로젝트의 Godot 내장 placeholder 자산 카탈로그.

## 스프라이트
- `Sprite2D` + 단색 PNG (16×16 / 32×32 / 64×64 그리드)
- `ColorRect` (즉석 색상 박스, 디버그·UI 프로토타입)
- 키컬러로 역할 구분(아군 파랑, 적 빨강 등)

## 타일맵
- `TileMap` + `TileSet` (단색 타일 또는 격자 텍스처)
- 격자 크기는 16/32/64 중 프로젝트 픽셀 해상도에 맞춤

## UI
- `Control` + `ColorRect` 박스 (HUD 슬롯·메뉴 프로토타입)
- `Label` 폰트는 Godot 기본 (커스텀 폰트는 사용자 위임)

## 애니메이션
- `AnimationPlayer` + frame 변경 또는 `AnimatedSprite2D`
- 단색 스프라이트 시트로 4~8 프레임 단순 애니메이션

## 카메라·라이트
- `Camera2D` (필수)
- `CanvasModulate` (전체 색조), `Light2D` (선택)

## 사용자 위임 트리거
다음은 ② 사용자 위임 단계로:
- 캐릭터 픽셀아트·일러스트
- 타일셋·배경 아트
- 정교한 애니메이션 시퀀스
- 사운드·음악
