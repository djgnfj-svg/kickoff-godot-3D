# asset-pipeline: 3D placeholder 카탈로그

3D 프로젝트의 Godot 내장 placeholder 자산 카탈로그.

## 메시
- `MeshInstance3D` + `BoxMesh` (캐릭터·박스 배치)
- `MeshInstance3D` + `SphereMesh` (구체·발사체)
- `MeshInstance3D` + `CylinderMesh` (기둥·통)
- `MeshInstance3D` + `CapsuleMesh` (캐릭터 기본형)
- `MeshInstance3D` + `PlaneMesh` (바닥·벽)

## 머티리얼
- `StandardMaterial3D` 기본 albedo (단색) — 키컬러로 역할 구분(아군 파랑, 적 빨강 등)

## 텍스처
- 단색 또는 체크보드 (UV 검증용)

## 애니메이션
- `AnimationPlayer` + 단순 transform 트랙 (translation/rotation)

## 라이트
- `DirectionalLight3D` 1개 + `WorldEnvironment` 기본 sky로 시작

## 사용자 위임 트리거
다음은 ② 사용자 위임 단계로:
- 캐릭터 모델 (스킨 메시·리깅)
- 환경 에셋 (props, 자연물)
- 사실적 머티리얼·노멀맵
- 사운드·음악
