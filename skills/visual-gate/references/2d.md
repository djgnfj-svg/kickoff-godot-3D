# visual-gate: 2D 시각 도식 카탈로그

2D 프로젝트에서 사용 가능한 CSS 클래스·보조 도식 패턴.

## CSS 클래스 (2D 적합)
- `scene-mockup` (공용, 16:9 실제 장면 흉내, 픽셀아트면 정수 스케일)
- `tile-grid` — 타일맵 격자 구조
- `node-graph` — Node2D 중심 씬 트리
- `silhouette` — 2D 캐릭터/스프라이트 실루엣
- `hud-slot` (공용)
- `timeline-bar` (공용)

## 보조 도식 패턴 예시
- 픽셀 해상도 결정 → 해상도 박스(예: 320×180)와 16:9 스케일링 도식
- 타일맵 레이어 → tile-grid + 레이어 깊이(배경/지형/오브젝트/UI) 표시
- 사이드뷰 패럴랙스 → 레이어 N개의 평행 이동 비율 도식
- 씬 트리 → node-graph (Node2D / Camera2D / TileMap / Sprite2D)

## 픽셀아트 주의
- 정수 스케일만 사용 (1x/2x/3x), 부분 스케일 시 흐려짐
- `texture_filter = Nearest` 가정

## 레퍼런스 게임 인용 시
2D 시점·아트 스타일이 비슷한 게임 2~3개 (탑다운/사이드뷰/쿼터뷰 등 같은 시점 우선)
