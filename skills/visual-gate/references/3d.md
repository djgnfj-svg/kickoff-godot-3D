# visual-gate: 3D 시각 도식 카탈로그

3D 프로젝트에서 사용 가능한 CSS 클래스·보조 도식 패턴.

## CSS 클래스 (3D 적합)
- `scene-mockup` (공용, 16:9 실제 장면 흉내)
- `camera-diagram` — 카메라 위치·각도·FOV 도식
- `depth-axis` — 카메라-피사체 깊이 표시
- `node-graph` — Node3D 중심 씬 트리
- `silhouette` — 3D 캐릭터 실루엣
- `hud-slot` (공용)
- `timeline-bar` (공용)

## 보조 도식 패턴 예시
- 1인칭 카메라 → camera-diagram (위치·yaw/pitch·FOV)
- 3인칭 어깨너머 → camera-diagram + depth-axis
- 탑다운 3D → camera-diagram (top view) + node-graph (Node3D 트리)
- 깊이감 결정 → depth-axis 단독

## 레퍼런스 게임 인용 시
3D 시점·카메라 동선이 비슷한 게임 2~3개를 명시 (장르가 달라도 OK)
